Number Guessing Game (Java)

A simple console-based game where the user tries to guess a randomly generated number between 1 and 100.

Features

* Random number generation within a defined range
* User input using Scanner
* Feedback on each guess (too high / too low)
* Tracks number of attempts until correct guess

How to Run

1. Compile the program:
    javac RandomNumberGuessGame.java
2. Run the program:
    java RandomNumberGuessGame

Concepts Used

* Variables and data types
* Conditional statements (if-else)
* Loops (do-while)
* Random number generation
* User input handling

What I Learned

* How to build a simple interactive program in Java
* Using loops and conditionals to control program flow
* Handling user input and generating random values
